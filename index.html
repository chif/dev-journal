◊ Investigation: Step-Up Catapult Bug

Walking into small physics objects (e.g. anvil on its side, ~33 cm < MaxStepHeight 40 cm) launches the character upward at ~18 m/s.

  1. Floor sweep detects object top as walkable floor
  2. ModifyContacts disables all contacts with the ground particle below step threshold
  3. CharacterGroundConstraintSolver closes the full gap in one iteration via both CP and DP accumulators
  4. With contacts disabled, nothing resists the resulting impulse — character catapults

Epic fixed this in 558c0c69b005 (2025-10-21) but the commit only exists in ue5-main — never backported to any release branch.
